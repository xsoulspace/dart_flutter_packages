# xsoulspace_vkplay_js

Generated `dart:js_interop` bindings and Dart wrappers for VK Play browser SDK (`iframeApi`).

## Exports

- `package:xsoulspace_vkplay_js/xsoulspace_vkplay_js.dart` - high-level wrapper API.
- `package:xsoulspace_vkplay_js/raw.dart` - low-level generated bindings.

## Generation

```bash
dart run tool/generate.dart
dart run tool/generate.dart --check
```

The generator tracks a lock file, SDK/runtime fixtures, declaration snapshots, and API diffs.
