# xsoulspace_platform_core_interface

Core contracts for unified platform adapters.

## Included API

- `PlatformId`
- `PlatformCapability`
- `PlatformClient`
- `PlatformInitOptions`
- `PlatformInitResult`
- `PlatformEvent`
- `PlatformException`
- `MissingPlatformCapabilityException`

Use this package as the base dependency for capability interfaces and adapters.
