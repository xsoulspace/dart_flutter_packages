# xsoulspace_monetization_ads_yandex

Yandex-specific ads monetization implementation for xsoulspace projects.

## Features

- Yandex ads provider integration layer
- Adapter implementation built on xsoulspace ads interfaces
- Ready for platform wiring in Flutter apps

## Installation

```yaml
dependencies:
  xsoulspace_monetization_ads_yandex: ^0.0.1
```
