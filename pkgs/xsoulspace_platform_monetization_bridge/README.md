# xsoulspace_platform_monetization_bridge

Compatibility export layer for split bridge packages.

Use this package only if you want one import that includes both:

- `xsoulspace_platform_purchases_bridge` (pure Dart)
- `xsoulspace_platform_ads_bridge` (Flutter)

## Re-exported API

- `PurchasesCapability`
- `PurchasesCapabilityAdapter`
- `NoopPurchaseProvider`
- `NoopPurchasesCapability`

- `AdsCapability`
- `AdsCapabilityAdapter`
- `NoopAdProvider`
- `NoopAdsCapability`
