# xsoulspace_crashhunter

Crash handling integration package for xsoulspace applications.

## Features

- Centralized crash reporting integration points
- Foundation hooks for app-level crash collection
- Designed to work with other xsoulspace observability packages

## Installation

```yaml
dependencies:
  xsoulspace_crashhunter: ^0.1.0
```

## Usage

Import the package and wire it into your app startup flow according to your
crash reporting pipeline.
