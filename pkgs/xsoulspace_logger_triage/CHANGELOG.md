# Changelog

All notable changes to this package will be documented in this file.

## 1.0.0-beta.0

- Initial beta release.
- Added `IssueTriageSink` for grouping/deduplication and escalation.
- Added deterministic fingerprint generation and token normalization.
- Added issue group model with status lifecycle and priority score.
- Added streaming and snapshot APIs for top issues.
- Added tests for normalization, dedup window, and escalation behavior.
